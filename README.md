# 리엑트 리덕스

## 참고
 - https://react-redux.js.org/introduction/quick-start
 - https://blog.naver.com/getinthere


## 설치
- yarn create react-app redux-app
- cd redux-app
- yarn add redux react-redux
- yarn add redux-devtools-extension

## 환경설정
- .prettierrc 파일 생성
```json
{
  "singleQuote": true,
  "semi": true,
  "useTabs": false,
  "tabWidth": 2,
  "trailingComma": "all",
  "printWidth": 80
}
```

- format on save 활성화
